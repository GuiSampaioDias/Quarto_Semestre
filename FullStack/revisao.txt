5- Etapas para se criar um Microserviço
-Desenvolver o código do microserviço
-Criar um Dockerfile
-Construir a imagem Docker
-Rodar o container a partir da imagem

Desenvolver o código do microserviço: Escrever o código que implementa a lógica do microserviço usando a linguagem de programação escolhida.
Criar um Dockerfile: Especificar um arquivo Dockerfile que define o ambiente necessário para rodar o microserviço. Isso inclui a imagem base, dependências e comandos necessários para construir a aplicação.
Construir a imagem Docker: Usar o Dockerfile para construir uma imagem Docker, que é uma versão empacotada e imutável do seu microserviço.
Rodar o container a partir da imagem: Criar e executar um container baseado na imagem Docker construída. Isso isola o microserviço e garante que ele será executado de forma consistente em diferentes ambientes.
